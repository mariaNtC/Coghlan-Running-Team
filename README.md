# Coghlan Running Team — Sitio Web

Proyecto final del curso de JavaScript en CoderHouse.

## Descripción

Sitio web para el equipo de running **Coghlan Running Team (CRT)**, con tienda online, galería de fotos, sección de eventos y formulario de contacto.

## Tecnologías

- HTML5 semántico
- SCSS (arquitectura modular con partials)
- JavaScript ES6+ (clases, fetch, localStorage)
- Bootstrap 5.3
- EmailJS (formulario de contacto sin backend)
- Toastify JS + SweetAlert2

## Funcionalidades

- Tienda con carrito de compras persistente (localStorage)
- Tres tipos de producto: membresías, indumentaria (con selección de talle) y artículos
- Galería de fotos con carrusel responsive
- Sección de eventos
- Formulario de contacto funcional con EmailJS
- Animaciones con AOS y Animate.css
